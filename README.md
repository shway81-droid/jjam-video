# 🎬 짬짬이 영상

**애매하게 남은 수업 시간, 뭐 틀지?** — 시간·주제·학년으로 1초 만에 찾는 선생님 전용 영상 아카이브.

[짬짬이 교실](https://shway81-droid.github.io/jjam/)의 보드게임풍 디자인과
[자투리타임](https://jaturi.me/)의 영상 큐레이션 콘셉트를 합친 **순수 정적 사이트**입니다.
설치·로그인 없이 브라우저에서 바로 동작하고, 서버가 필요 없습니다.

## ✨ 기능

- **⏱ 시간·👥 학년·🏷 주제 필터** — 남은 시간 안에 소화 가능한 영상만 1초에 추림
- **🔍 검색** — 제목·소개·주제 키워드로 바로 찾기 (필터·저장함·계기교육과 겹쳐 적용)
- **📅 계기교육 달력** — 기념일별 전용 영상 안내. 음력 명절(설날·추석·부처님오신날)은
  `app.js`의 `LUNAR_MD` 표(한국천문연구원 음양력 데이터, ~2036년)로 매년 자동 반영
- **🎲 룰렛으로 골라줘** — 현재 필터 안에서 랜덤 추천 후 바로 재생
- **🍿 오늘의 추천** — 첫 화면 추천 카드 + "추천 새로 받기"
- **⭐ 저장함** — 마음에 든 영상 담아두기 (브라우저 `localStorage`, 로그인 없음)
- **🧩 이렇게 써보세요** — 영상마다 수업 활용 아이디어 3가지
- **📺 전자칠판 최적화** — 1280px 이상 대화면에서 글씨·카드 자동 확대, 전체화면 버튼

## 🗂 구조

```
index.html        런처 (히어로·필터·그리드·모달)
style.css         보드게임풍 테마 (펠트그린·우드·골드·크림카드)
app.js            필터·룰렛·저장함(localStorage)·모달 재생
data/videos.json  영상 큐레이션 데이터 (단일 소스)  ← 여기만 수정하면 됨
favicon.svg
```

## ➕ 영상 추가·수정 방법

`data/videos.json` 배열에 항목을 추가하면 끝입니다. (빌드 불필요)

```json
{
  "id": "unique-slug",                        // 고유 식별자 (영문·숫자·하이픈)
  "title": "지식채널e - 나는 정말 의지력이 부족한 걸까",
  "youtubeId": "Q4cC7WrBk9s",                 // 유튜브 URL의 v= 뒤 11자리
  "topic": "인성",                             // 주제 (아래 목록 참고)
  "grade": ["중학년", "고학년"],                // 저학년 / 중학년 / 고학년 중 해당되는 것
  "minutes": 5,                               // 영상 길이(분)
  "description": "영상 한 줄 소개",
  "ideas": ["활용법 1", "활용법 2", "활용법 3"]   // 수업 활용 아이디어 3가지
}
```

- **youtubeId**: `https://www.youtube.com/watch?v=Q4cC7WrBk9s` → `Q4cC7WrBk9s`
  (`https://youtu.be/Q4cC7WrBk9s`의 뒷부분과 동일). 썸네일·재생이 자동 연결됩니다.
- **topic 권장 목록**: 인성, 진로, 건강·보건, 생명존중, 세계시민, 가족, 역사,
  다문화, 학교폭력예방, 힐링·재미, 경제, 인권, 독도, 민주시민, 예술·문화, 스포츠, AI교육,
  미디어리터러시, 장애인식개선, 안전, 과학·탐구, 학급자치, 통일, 환경생태
  *(목록에 없는 주제를 써도 필터에 자동 추가됩니다.)*
- `youtubeId`가 `"SAMPLE"`이면 예시 카드로 표시됩니다. 실제 ID로 바꾸세요.

## ▶️ 로컬 실행

`fetch`로 JSON을 읽으므로 파일을 직접 여는 대신 로컬 서버로 엽니다.

```bash
python -m http.server 8000
# http://localhost:8000
```
